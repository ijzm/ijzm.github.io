Only use this method to browse websites you trust.
Please read the full guide before doing anything.
This guide has only been tested on Windows.

Flash Instalation FAQ

1. Install Pale Moon

Pale Moon is an Open Source free web browser that will keep supporting Flash Player. It is based off an old version of Firefox

Pale Moon for Windows Pale Moon for Linux

2. Install Flash Player

If you already had Flash installed you may need to uninstall it first.

The Internet Archive Website hosts official downloads for old versions of Flash here

Flash Player Installers Archive

You will probably want to download the newest version, that doesn't contain the January 12th timebomb code, in this case version 32.0.0.363. But you may want some older versions depending on specific games. This zip files contains the installer for all the different operative systems. But if you go to the "View Contents" link, then you can download just the installer for your own Operative System.

Recommended Version of Flash

Direct Downloads:

Windows - [Version]/flashplayer[Version]_win.msi Mac - [Version]/flashplayer[Version].dmg Linux32 - [Version]/flashplayer[Version]linux.i386.tar.gz Linux64 - [Version]/flashplayer[Version]x86_64.tar.gz

3. Enjoy!

Open Pale Moon and go to your favorite Website to test. Everything should work just fine!
